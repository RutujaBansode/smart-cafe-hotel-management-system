# Smart Cafe & Hotel Management System

A Java Web Application developed using JSP, Servlets, JDBC, MySQL, and Apache Tomcat.

## Features

- User Authentication
- Role-Based Access (Admin & Staff)
- Inventory Management
- Order Processing
- Billing System
- Low Stock Alerts
- Session Management

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- HTML
- CSS
- JavaScript

## Project Structure

src/
├── main/
│   ├── java/
│   └── webapp/

## Database Setup

1. Install MySQL.
2. Create a database.
3. Import the SQL script.
4. Update database credentials in the project.
5. Run the project on Apache Tomcat.
